Challenges using GitHub

GitHub has been an amazing tool to be able to work and collaborate together as a group and while not always having to meet up physically, or not having to transfer our codes to each other using a memory stick or some other way.

Though as practical and useful as GitHub has been, we've had couple issues setting up the GitHub collab, and faced a lot of merge conflicts due to not pulling and pushing our updates often enough.

Issues including:

  • Complete paragraphs of code going missing
  • Duplication of code elements
  • Someone being behind 5 updates and not getting any pull requests
  • Pushing an old version so everyone who pulled it went back
